# Greedy Algorithms

This folder contains implementations and notes for greedy algorithms.

## Topics Covered

- **Classic Greedy Problems**
  - Activity Selection Problem
  - Fractional Knapsack
  - Job Sequencing Problem
  - Huffman Encoding
  - Minimum Spanning Tree (Kruskal's, Prim's)

- **Interval Problems**
  - Interval Scheduling
  - Interval Partitioning
  - Weighted Interval Scheduling
  - Merge Intervals

- **Array Problems**
  - Minimum Platforms
  - Jump Game
  - Gas Station
  - Container with Most Water

- **Graph Greedy Algorithms**
  - Dijkstra's Shortest Path
  - Prim's MST
  - Kruskal's MST

- **String Problems**
  - Lexicographically Smallest String
  - String Compression

- **Mathematical Greedy**
  - Egyptian Fractions
  - Minimize Cash Flow
  - Coin Change (when greedy works)

- **Scheduling Problems**
  - Task Scheduling
  - Minimize Lateness
  - Job Sequencing with Deadlines

## Usage

Add your implementations and code snippets here with proper documentation.

## Important Note

Remember: Greedy algorithms don't always give optimal solutions. Always verify that the greedy choice property and optimal substructure hold for your problem!
