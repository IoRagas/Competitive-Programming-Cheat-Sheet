# Dynamic Programming

This folder contains implementations and notes for dynamic programming problems.

## Topics Covered

- **Classical Problems**
  - Fibonacci Numbers
  - Climbing Stairs
  - Coin Change
  - Rod Cutting
  - Longest Common Subsequence (LCS)
  - Longest Increasing Subsequence (LIS)
  - Edit Distance
  - Longest Palindromic Subsequence

- **Knapsack Problems**
  - 0/1 Knapsack
  - Unbounded Knapsack
  - Bounded Knapsack
  - Subset Sum
  - Partition Problem

- **Matrix Chain Multiplication**
  - Optimal Parenthesization
  - Palindrome Partitioning

- **String DP**
  - Word Break
  - String Interleaving
  - Wildcard Matching
  - Regular Expression Matching

- **DP on Trees**
  - Tree Diameter
  - Tree Matching
  - Subtree DP

- **DP on Graphs**
  - Shortest Path with DP
  - Travelling Salesman Problem (TSP)
  - Hamiltonian Path

- **Digit DP**
  - Count Numbers with Property

- **Bitmask DP**
  - Subset DP
  - Assignment Problem

- **Optimization Techniques**
  - Space Optimization
  - Convex Hull Trick
  - Divide and Conquer Optimization
